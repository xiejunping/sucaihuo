## 使用
下载源码下来，进入目录

* 1、npm install

* 2、config/user.json 填写你的用户名与密码，一个对象一组，
如果没有账号，申请吧

* 3、node index.js 
会打出key , 20,-1

20 是代表签到成功
-1 是重复签到
